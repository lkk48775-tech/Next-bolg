## 动态面包屑目录

> 面包屑用于展示当前页面所在层级，让用户知道自己在哪里，也能快速返回上级。

### 适用场景

- 后台管理
- 文章详情
- 多级菜单
- 动态路由

### 技术点总览

| 技术点 | 主要作用 |
| --- | --- |
| 配置面包屑数据 | 可以从路由配置或菜单配置生成面包屑。 |
| 渲染面包屑 | 根据匹配结果渲染链接列表。 |
| 详情页动态标题 | 详情页标题可以从文章数据里拿。 |

### 分批介绍

#### 配置面包屑数据

可以从路由配置或菜单配置生成面包屑。

<CodeWindow
  lang="js"
  code={`
const menus = [
  { path: '/', title: '首页' },
  { path: '/articles', title: '文章' }
]
`}
/>

#### 渲染面包屑

根据匹配结果渲染链接列表。

<CodeWindow
  lang="jsx"
  code={`
function Breadcrumb({ items }) {
  return items.map((item) => (
    <Link key={item.path} to={item.path}>{item.title}</Link>
  ))
}
`}
/>

#### 详情页动态标题

详情页标题可以从文章数据里拿。
